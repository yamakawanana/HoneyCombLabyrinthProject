【Honeycomb Labyrinth】

アヒルさんとクマさんが蜂の巣に侵入し、ハチに刺されないように気を付けながら、制限時間以内にゴールまでハチミツを取りに行くゲームです。
アヒルさんを操作すると、クマさんが後から付いて来ます。
２匹ともゴールに到着するとクリアです。
アヒルさんがハチに刺されるか、制限時間以内にゴールできないとゲームオーバーになります。

【操作方法】

右移動：→キー
左移動：←キー
右上移動：→キーと↑キー同時押し
右下移動：→キーと↓キー同時押し
左上移動：←キーと↑キー同時押し
左下移動：←キーと↓キー同時押し
